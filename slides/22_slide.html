Whats this noise?

Random is too random

To create chaotic but pleasant images we often do not want to use random.

  • "Pure" randomness looks unnatural
  • We want something unpredictable, yet organic looking
  • Thats the noise!

🢂 We want to smoothly interpolate between the random peaks

Using noise

Lets play with this, can we use this noise to

  • Animate a shape
  • Change the coordinate system
  • Combine multiple noise controlled parameters
