Shapes and sights – Toolbox

Step functions

  • step(float edge, float x) – takes an input float x and an edge value. If the input value is less than edge, output of the function will be 1, else 0.
  • smoothstep(float edge1, float edge2, float x) – same as step, but instead of one edge it takes two edges, between which output will be smoothly interpolated. Meaning output – 0, if the value is less than edge1. Then output smoothly rises to 1 between edge1 and edge2. Then stays 1 after value continues rising past edge2.
Try commenting and un-commenting the lines for each step function in following shader. Notice how hard edge turns into smooth gradient.

🢂 Hint: Pay attention how values that are put as edges into step functions, map to our coordinate system. Try tweaking them.

🢂 Hint: Step, smoothstep and many other functions can also take other types as arguments – vec2, vec3, and will interpolate each value from vector separately.
